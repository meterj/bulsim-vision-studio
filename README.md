# 🏮 BULSIM VISION V2.0 (불심 비전)

> **"당신의 마음속 깨달음을 장엄한 비전으로 빚어내다"**

불심 비전(Bulsim Vision)은 사용자의 간절한 발원과 마음의 화두를 AI 기술(Gemini API)을 통해 성스러운 불교적 시각 예술로 승화시키는 프리미엄 디지털 법보 스튜디오입니다.

![App Screenshot](https://via.placeholder.com/1200x600/1a1a1a/gold?text=BULSIM+VISION+V2.0)

## ✨ 주요 기능 (Key Features)

### 🧘‍♂️ 범불교적 지혜 해석 (Universal Dharma Logic)
특정 종파에 치우치지 않는 보편적인 불교 미학을 바탕으로, 사용자의 의도를 분석하여 최적의 상징물과 색채를 제안합니다.

### 🎨 고도화된 예술 양식 (Artistic Styles)
- **전통 탱화 (Taenghwa)**: 장엄한 불화의 기법으로 발원 내용을 시각화합니다.
- **선묵화 (Zen Ink Wash)**: 여백의 미와 절제된 붓 터치로 깨달음의 순간을 담습니다.
- **불상 조각 (Statue)**: 숭고한 자비의 미소를 정교한 조각 양식으로 표현합니다.

### 📜 디지털 법보 리포트 (Dharma Report)
생성된 비전에 담긴 상징적 의미와 불교적 가르침을 정식 HTML 리포트로 간직할 수 있습니다.

### 📁 고품질 이미지 저장 (Premium Download)
하드웨어 가속 캔버스 렌더링을 통해 고화질 JPG 파일로 깨달음의 비전을 소장할 수 있습니다.

## 🚀 시작하기 (Getting Started)

1.  **Repository 클론**:
    ```bash
    git clone https://github.com/meterj/bulsim-vision-studio.git
    ```
2.  **의존성 설치**:
    ```bash
    npm install
    ```
3.  **개발 서버 실행**:
    ```bash
    npm run dev
    ```
4.  **API 키 설정**: 앱 우측 상단의 심인(心印) 아이콘을 클릭하여 Gemini API 키를 입력하세요.

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS (Premium Dark Theme)
- **AI Engine**: Google Gemini API (v1beta)
- **Animations**: Framer Motion, Lucide React
